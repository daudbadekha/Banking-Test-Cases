# 🏦 Banking Test Cases

## 📌 Overview

This repository contains sample manual test cases for core banking modules. The content is designed for learning, practice, and portfolio purposes using generic banking scenarios. No confidential or proprietary information is included.

---

## 🎯 Scope

The repository covers manual testing scenarios for:

- Login & Authentication
- Customer Onboarding
- CASA (Current & Savings Account)
- Fund Transfer
- NEFT
- RTGS
- IMPS
- UPI
- Fixed Deposit
- Loan Processing
- Standing Instructions

---

## 🧪 Test Design Techniques

- Positive Testing
- Negative Testing
- Boundary Value Analysis
- Equivalence Partitioning
- Input Validation
- Business Rule Validation
- Functional Testing
- Regression Testing
