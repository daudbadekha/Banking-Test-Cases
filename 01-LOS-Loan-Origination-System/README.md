# 🏦 Loan Origination System (LOS)

## Overview

Loan Origination System (LOS) is responsible for managing the complete loan application lifecycle, starting from customer application until loan approval and disbursement.

This module focuses on validating business rules, customer eligibility, documentation, workflow approvals, and loan creation.

---

# Business Flow

Customer Registration

↓

Loan Application

↓

KYC Verification

↓

Document Upload

↓

↓

Credit Assessment

↓

Approval / Rejection

↓

Sanction Letter

↓

Loan Creation

↓

Loan Disbursement
